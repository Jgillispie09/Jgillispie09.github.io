A Padding, Border, Margin Cheat Sheet

by

Joshua Gillispie

Command Name Description
Padding, -top, -right, -bottom, -left Padding Area between padding and the border
  • Can be set in length, percentage, initial, inherit
    • Length must be set in pixels, must be positive number
    • Percentage sets padding in percentage of width of element from 0 to 100 percent
    • Initial sets property to default value
    • Inherit allows element to inherit padding property from its parent element
  • Can adjust specific areas by using -top, -right, -bottom, -left
    • -top adjusts the amount of padding between the top of the content and the border
    • -right adjusts the amount of padding between the right side of the content and the border
    • -bottom adjusts the amount of padding between the bottom of the content and the border
    • -left adjusts the amount of padding between the left side of the content and the border
Border, -left, -top, -bottom, -right Border Area between padding and margin
  • Can set border properties such as width, style, and color
    • Width can be adjusted with border-width
    • Set style of border using border-style
    • Set color of border using border-color
    • To reset a property to default use initial
    • To use a property inherited by a parent element use inherit
Margin, -left, -top, -bottom, -right Margin Space around elements outside of borders
  • Can set margin space around border
    • Length can be adjusted using pixels
    • Adjust margin size by using percentage '%'
    • Let the browser determine the margin with auto
    • Reset to default by using initial
    • Inherit the properties of a parent element with inherit
