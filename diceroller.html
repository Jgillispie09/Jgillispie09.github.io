Dice Roller






Versatile Dice Roller

Are you a DnD player in need of a reliable dice roller? Look no further! I created this web app using a combination of HTML, CSS, and JavaScript to help adventurers roll their dice with ease.

With HTML, I designed the app's interface, including buttons, labels, and input fields to make it easy for users to interact. CSS allowed me to shape the layout and add a thematic background image that brings the app to life.

The heart of the app lies in JavaScript. When a user selects a die (d4, d6, d8, d10, d12, or d20) and the number of dice to roll, JavaScript takes the user's input and passes it into a function that generates random numbers for each die. For example, if the user rolls 3 d4s, the app will roll each die individually and calculate the total of the results.

This app is perfect for any adventurer looking to streamline their dice rolls and focus on the adventure ahead!
