WebCrawler (GameCrawler)

The python file can be downloaded here.

Program Overview:

This Python program is a web crawler designed to scrape four game review websites for data on trending games and their reviews. The primary purpose is to gather and compare review information across different platforms, offering insights into how trending games are perceived by various review sources.

Modules Used:

* Requests: Sends HTTP requests to web servers and receives response codes to ensure servers are accessible.
* BeautifulSoup (bs4): Parses the HTML content of each website, acting as the core parser for the web crawler.
* Pandas: Creates dataframes to structure and manipulate the scraped data efficiently.
* Datetime: Adds timestamping to generated Excel files for proper date and time tracking.
* Openpyxl (load_workbook): Loads existing Excel files and saves new data into the workbook.

Python Concepts Utilized:

* Control Structures: while loops, if statements, and for loops to manage program flow.
* Data Handling: Lists, dataframes, and modules (import statements).
* Web Requests: .get() method for retrieving web pages.
* Data Parsing: BeautifulSoup to navigate and extract HTML content.
* ETL (Extract, Transform, Load): Gathers data, processes it into a readable format, and saves it to an Excel file.

How It Works:

* Website Requests: The program uses the Requests module to send HTTP requests to all four game review websites.
-- If all websites return a status code 200, the program proceeds.
-- If any website fails, an error message is displayed, and the program exits.
* Data Extraction: BeautifulSoup parses the HTML content of each website. The program identifies specific HTML elements containing review information. It then navigates through the HTML structure to extract game titles, ratings, review excerpts, and other key details. Extracted data from each website is stored in separate lists.
* Data Organization: The program converts the lists into Pandas dataframes, organizing the data in a tabular format. Variables are set for the workbook name, including the current date and time using Datetime.
Data Export to Excel: The program writes all the extracted data into an Excel file. It opens the workbook using Openpyxl and formats the columns to avoid squished data. The workbook is saved, and a success message is displayed in the console.

Key Features:

* Error Handling: Detects server issues early, preventing wasted processing time.
* Automated Formatting: Ensures the Excel file is readable and well-organized.
* Data Comparison: Allows for cross-platform analysis of game reviews.
